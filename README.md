# Autonomous Financial Research Agent (ARA-1)

## Overview
This repository contains the source code for the Autonomous Financial Research Agent (ARA-1) developed for QuantumEdge Research. It autonomously replicates the research workflow of a junior financial analyst using a Plan-and-Execute reasoning loop, a 3-layer memory system, and a multi-source synthesis engine.

## Environment Setup
1. Ensure Python 3.10+ is installed.
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -e .
   ```
4. Copy `.env.example` to `.env` and fill in your API keys.
